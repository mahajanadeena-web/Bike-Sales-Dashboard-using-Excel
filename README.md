# Bike-Sales-Dashboard-using-Excel

Objective
To analyse customer demographics and behavioural patterns influencing bike purchases using Excel, and present findings through an interactive dashboard.

Dataset Overview
The dataset includes customer details such as:
•	Demographics: Age, Gender, Marital Status, Education, Region
•	Behavioural Data: Commute Distance, Purchase Status
•	Financial Data: Income

Data Preparation Steps
1.	Remove Duplicates
Ensured data integrity by eliminating duplicate customer entries.
2.	Find and Replace
Standardized inconsistent values (e.g., correcting typos in education levels or regions).
3.	Created Age Bracket Column
Segmented customers into:
•	Adolescent
•	Middle Age
•	Old
This enabled clearer trend analysis across age groups.

Dashboard Construction
1.	Pivot Tables
o	Aggregated purchase counts by age bracket, commute distance, and gender.
o	Calculated average income per purchase status.
2.	Charts
o	Bar Chart: Average Income vs. Purchase Status by Gender
o	Line Chart: Bike Purchases by Age Bracket
o	Line Chart: Bike Purchases by Commute Distance
3.	Slicers for Interactivity
•	Marital Status
•	Region
•	Education Level
These slicers allow dynamic filtering to explore how different segments behave.

Key Insights
•	Income Influence: Higher average income correlates with increased bike purchases, especially among males.
•	Age Trends: Middle-aged customers show the highest purchase activity.
•	Commute Patterns: Customers commuting 2–5 miles are most likely to purchase bikes.

Tools Used
•	Microsoft Excel 2013
•	Pivot Tables
•	Slicers
•	Conditional Formatting
•	Custom Age Bracketing

An interactive dashboard that helps identify target customer segments for marketing and sales strategy. This project demonstrates proficiency in Excel analytics, dashboard design, and data storytelling.
